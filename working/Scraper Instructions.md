Scraper Instructions

1. Double click the .exe file to run the program. The first time will take several minutes to set up, but after that it should only take 1-2 minutes to load
2. Put in the password, the encryption password and clint's username should be auto-filled
3. Click login. It should take a <30 seconds
4. Click scrape. Depending on how long it's been since you last updated the data, it could take a while, up to several hours
5. Click upload. This should be pretty quick, <1 minute
6. All done! You're good to close the program

Do that every 6 months or so to keep the data up to date

Notes:
* The first time it'll take a while to load, and possibly every time after that
* The "Encryption password" is the same as the password for the web program
